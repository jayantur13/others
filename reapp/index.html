Open Source Libraries

This app uses the following open-source libraries and tools:

  • Bootstrap 5 - For responsive design and layout.
  • FMStream - For radio station urls.
  • InternetRadio - For radio station urls.
  • Lorem Picsum - For fetching placeholder album artwork.
  • oktoast - For showing toast like native android.
  • path_provider - For getting file path.
  • just_audio - For the audio player.
  • just_audio_background - For audio background play.
  • http - For all networking purpose.
  • permission_handler - For asking user necessary device permissions.
  • url_launcher - For dealing with intents.
  • package_info_plus - For showing build number, version etc.
  • shared_preferences - For saving local small data.
  • cupertino_icons - For default icons android/iphone.
  • background fetch - For scheduling background tasks.
  • flutter local notifications - For sending local notifications by app itself.
  • Network Checker - For checking internet connectivity.
  • google mobile ads - For generating and showing ads to user.
  • mini music visualiser - For showing animated bars on the played station.
  • marquee - For html equivalent marquee.
  • flutter_lints - For good coding practise.
  • flutter - For the overall designing and development.
