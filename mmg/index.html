Open Source Libraries

This game uses the following open-source libraries and tools:

  • Bootstrap 5 - For layout and responsiveness.
  • Flutter - For base of the game.
  • Flame - Game engine.
  • Flame Audio - For audio inside game.
  • Flame Splash Screen - For splash branding.
  • oktoast - For native like android toast.
  • animate_do - For animation in game.
  • url_launcher - For android intents.
  • google_mobile_ads - For mobile ads (AdMob).
  • shared_preferences - For storing data in device.
  • share_plus - For send action intent android.
  • path_provider - For getting file path.
  • permission_handler - For asking user necessary device permissions.
  • package_info_plus - For showing build number, version etc.
  • Icon Kitchen - App Logo.
  • background fetch - For scheduling background tasks.
  • flutter_lints - For good coding practise.
  • awesome_notificatios - For local notifcations.
  • Cyberwave-Orchestra - Background music
  • Mixkit - Sound Effects
